# 📊 Bank Marketing Inspection Analysis

## 📌 Project Overview

This project focuses on analyzing a **Bank Marketing Dataset** to understand customer behavior and predict whether a client will subscribe to a term deposit. The analysis is performed using **Python** and key data analysis libraries.

The goal is to extract insights, identify patterns, and support decision-making using data-driven approaches.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing
* Handle missing and "unknown" values
* Conduct exploratory data analysis (EDA)
* Visualize trends and relationships
* Analyze factors influencing customer subscription

---

## 🛠️ Tech Stack

* **Python**
* **NumPy**
* **Pandas**
* **Matplotlib**

---

## 📂 Dataset Description

The dataset contains information about bank clients and marketing campaigns, including:

* **Age** – Client's age
* **Job** – Type of job
* **Marital** – Marital status
* **Education** – Education level
* **Default** – Credit default
* **Housing** – Housing loan status
* **Loan** – Personal loan status
* **Contact** – Communication type
* **Duration** – Call duration
* **Campaign** – Number of contacts during campaign
* **Previous** – Number of previous contacts
* **Poutcome** – Outcome of previous campaign
* **y** – Target variable (Subscribed: Yes/No)
